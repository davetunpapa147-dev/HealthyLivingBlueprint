Docker Deployment

The entire stack ships as a Docker Compose configuration. All services are pre-wired with environment variable templating, health checks, and volume mounts.

docker-compose.yml 
version: '3.9'

services:
  nextjs:
    build: ./nextjs-app
    ports:
      - "3000:3000"
    environment:
      - NEXT_PUBLIC_API_URL=http://laravel:8000

  laravel:
    build: ./laravel-api
    ports:
      - "8000:8000"
    depends_on:
      - postgres
      - redis

  postgres:
    image: postgres:16
    volumes:
      - pgdata:/var/lib/postgresql/data

  redis:
    image: redis:latest

volumes:
  pgdata:

API Endpoints

RESTful JSON API under /api/v1/ — versioned from day one. All endpoints return consistent envelope responses.

Method Endpoint Description
POST /api/v1/login Authenticate user, return Sanctum token
POST /api/v1/register Register new patient or provider
GET /api/v1/articles Paginated health article feed
POST /api/v1/appointments/book Book a new appointment slot
POST /api/v1/ai/chat AI assistant endpoint (streaming)
